# Коммерческое предложение для клиента

![Preview](preview.png)

Профессиональный шаблон коммерческого предложения для финансовых консультантов и wealth-менеджеров. Включает персонализированное резюме клиента, интерактивные карточки услуг с раскрывающимся содержимым, три модели обслуживания на выбор, структуру тарифов и детальный процесс онбординга. Идеально подходит для презентации комплексных финансовых услуг состоятельным клиентам.
